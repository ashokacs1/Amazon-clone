# Amazon Clone - UI/UX Practice

A front-end clone of the **Amazon.com** homepage and footer, built to master the art of complex layouts and responsive web design.

This project is a pixel-perfect recreation of the Amazon web interface. The goal was to practice semantic HTML and modern CSS techniques to handle dense information and multi-layered navigation.

### Key Features

* **Navigation Bar:** Includes the search bar, account lists, and location selector.
* **Hero Section:** A responsive image slider/carousel effect.
* **Product Grids:** Flexbox-based cards showing various product categories.
* **Detailed Footer:** Multi-column link sections, language/currency selectors, and the copyright panel.
* **Responsive Design:** Optimized for different screen sizes using Media Queries.
